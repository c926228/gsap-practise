# gsap-practise (重製中)

本專案用來練習vue3, gsap, unocss
用以上技術重製日本溫泉網站首頁動畫效果，未包含RWD。

- [DEMO](https://c926228.github.io/gsap-practise/)。
- 參考網站： [るるる温泉](https://ru-ru-ru.com/)

## 使用技術
---
- Gsap 實踐動畫效果
- Vue3 + Vite 建構專案
- UnoCSS + scss 完成畫面樣式
